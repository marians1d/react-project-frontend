## General info
This project is front-end application. The main purpose is to order custom furniture with ideas and image ideas.

## Technologies
Project is created with:
* React
* React Router
* React Redux
* Validator 
* Font Awesome
* Class Names

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
